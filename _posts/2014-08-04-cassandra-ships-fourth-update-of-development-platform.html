Additional the release Version 11.4 offers you a lot of new features. title: Cassandra ships fourth update of Development Platform --- Cassandra Team is pleased to announce the immediate availability of a new Cassandra software update. This is a maintenance release for the 11 series of Cassandra. Additional the release Version 11.4 offers you a lot of new features.

Main advantages and achievements of the new release are:

  • More than 30 new coding tutorials with example graphs
  • Performance boost of the Data-Flow for pipelined processing of images
  • Improvement of the visualization features of the Layout View
  • Availability of additional Download versions for 64bit OS architecture and Visual Studio 2013 Environment
  • Migration to OpenCV Version 2.4.9

    • Try these and other exciting features by downloading Cassandra 11.4!
