--- date: 2015-07-20 11:33:12 layout: post permalink: opencv-manager-beta-testing.html published: true short: We invite you to take part in OpenCV Manager for Android beta testing. title: OpenCV 2.4+3.0 Manager for Android beta testing starts --- We invite you to take part in OpenCV Manager for Android beta testing. Some features have been added in this release:

  • added support for both 2.4 and 3.0 libraries
  • added support for 64-bit platforms (3.0 only)
  • dropped armeabi-v7a without NEON
  • dropped native camera support (existing OpenCV 2.4-based native or Java applications, as well as new OpenCV 3.0-based Java applications should work well)

To enter beta testing, follow these steps:

  • Subscribe to android-opencv@googlegroups.com Google group (https://groups.google.com/d/forum/android-opencv)
  • Visit one of the links below and push "Become a tester" button to activate beta-channel for corresponding application:
    • OpenCV Manager - https://play.google.com/apps/testing/org.opencv.engine
    • OpenCV 15 puzzle - https://play.google.com/apps/testing/org.opencv.samples.puzzle15
    • OpenCV Face Detection - https://play.google.com/apps/testing/org.opencv.samples.facedetect
  • Visit Google Play on your device and upgrade selected application or wait for the auto-upgrade
